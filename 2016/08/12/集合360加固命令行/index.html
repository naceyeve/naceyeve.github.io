集合加固命令

发表于 2016-08-12   |   分类于 Android Studio

##1.设置.bat文件

在C：\盘下新建basebat文件夹；新建txt文档，保存为jiagu.bat;配置环境变量，在path路径中新增c:\basebat;

jiagu.bat内容：

 
@echo on
rem 360加固存放目录
cd C:\Users\Administrator\Desktop\2026\360jiagubao_windows_64\jiagu\
rem 登陆360 帐号 和 密码
java\bin\java -jar jiagu.jar -login naceyeve 11111111
rem  配置签名信息
java\bin\java -jar jiagu.jar -importsign F:\KeyStores 123456 gradlemutli 123456 
rem  配置渠道信息（此处直接导入渠道模板）
java\bin\java -jar jiagu.jar -importmulpkg C:\Users\Administrator\Desktop\2026\360jiagubao_windows_64\jiagu\mode.txt
rem  加固 第一目录是待加固apk绝对路径；第二目录是已加固apk存放目录（新建了一文件夹存放output）
java\bin\java -jar jiagu.jar -jiagu F:\smartThree\trunk\SmartHome\app\build\outputs\apk\app-release.apk  F:\smartThree\trunk\SmartHome\output
##2.配置gradle文件 >在项目app文件夹下的build.gradle中修改，新增task; 
     project.afterEvaluate {        //在Release执行以后              
      tasks.getByName("assembleRelease"){        
        it.doLast{
         packageChannel('', '', '')        
        }     
       }   
     }
定义pachageChannel(x,x,x)方法 
 packageChannel(String versionName,String appName,String releaseApk){   
     try {      
          def stdout = new ByteArrayOutputStream()     
               exec {            //执行Python脚本//          
        workingDir   'C:\\basebat'       
        commandLine 'cmd','/c','jiagu.bat'       
         println 'hello world'      
        }        return stdout.toString().trim()   
     }    catch (ignored) { 
       return "UnKnown";   
 }

##3.编绎打包

双击assembleRelease，打包发布版本的apk，编译完成后在output文件夹下保存有加固后的apk
